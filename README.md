# RESO — Coffee Experience System

RESO is a conceptual brand project that explores coffee as both a physical product and a cultural experience.

The project connects origin, ritual, packaging, and visual storytelling into one unified system — where every cup reflects a place, a process, and a memory.

## 🌐 Live Site
[https://ZOZOBAR.github.io/reso-coffee/](https://zozobar.github.io/GD4-RESO/)

## 🧠 Concept

RESO is built on a dual perspective:

- **Inside** — emotion, memory, and personal ritual  
- **Outside** — origin, geography, and global coffee culture  

The system translates these ideas into a tangible experience through product design and visual language.

## 🧩 System Components

- **Coffee Machine** — a tool that bridges origin and daily ritual  
- **Origin Coffee Packaging** — communicates place, process, and identity  
- **Brew Cards** — transform information into a physical, collectible guide  

## 🎨 Design Language

The visual system is inspired by:

- Topographic lines (terrain & origin)
- Warm neutral tones (paper, milk, morning light)
- Forest green (land & coffee plant)
- Controlled amber accents (brewed coffee warmth)

## 📦 Refined Experience

The refined packaging system extends the brand into premium touchpoints:

- Machine box as a storytelling object  
- Brew card as a tactile experience  

The goal is to shift packaging from a functional container to a meaningful object.
